## Blog Platform — Spring Boot & Angular

### ⚙️ Installation
#### 1. Clone repository
```bash
git clone https://github.com/yasmine-draoui/blog-project.git
cd blog-project
```

#### 2. Backend — Spring Boot
```bash
cd backend
```
Configure database in:
```bash
application.properties
```
Run:
```bash
mvn spring-boot:run
```
#### 3. Frontend — Angular
```bash
cd frontend
```
#### Install dependencies:
```bash
npm install
```
Run app:
```bash
ng serve
```
